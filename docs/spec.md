# YepBuddy 앱 페이지별 명세서

> **문서 목적**: 기능 중심의 페이지별 명세서 (디자인 문서 아님)
> **최종 수정**: 2026-03-22
> **상태**: 초안 — 페이지별 상세 내용 추가 예정

---

## 공통 요소

### 하단 고정 알약바 (Pill Tab Bar)
- 항상 화면 하단에 고정
- 3개 탭: **일지** | **템포** | **프로틴**
- 선택된 탭은 주황(accent) 배경으로 강조
- 라우팅: 각 탭 클릭 시 해당 메인 페이지로 이동

---

## 📌 전체 화면 맵 (개발 순서)

```
Phase 1: 핵심 운동 플로우
├─ 1. 운동 기록 화면 (운동 세팅)
├─ 2. 카운트다운 → 운동중 화면 (운동 실행)
└─ 3. 운동 결과 화면 (운동 완료 후 확인)

Phase 2: 데이터 조회
├─ 4. 메인화면 - 요약 (대시보드)
│   ├─ 오늘의 운동 카드 → 7. 캘린더 페이지
│   │                       └─ 캘린더 항목 클릭 → 3. 운동 결과 화면
│   ├─ 세션 카드 → 3. 운동 결과 화면
│   ├─ 이번주 세션 "더보기" → 5. 세션 목록 화면
│   │                          └─ 세션 항목 클릭 → 3. 운동 결과 화면
│   └─ 운동 카드 → 1. 운동 기록 화면
│                   └─ 운동 시작 → 2. 카운트다운 → 운동중 화면
├─ 5. 세션 목록 화면 (전체 세션 히스토리)
├─ 6. 캘린더 페이지
│   └─ 캘린더 항목 클릭 → 3. 운동 결과 화면

Phase 3: 부가 기능
├─ 7. 운동 템포 페이지
├─ 8. 프로틴 페이지
└─ 9. 프로틴 상세 페이지

Phase 4: 위젯
└─ 10. 잠금화면 위젯
```

# Phase 1: 핵심 운동 플로우

---

## 1. 운동 기록 화면

| 항목 | 내용 |
|------|------|
| **진입 경로** | 메인화면 운동 카드 클릭 |
| **네비게이션** | ← 뒤로가기, 타이틀 "운동 기록" |

### 구성 요소

**① 운동 부위 선택**
- 칩(Chip) 형태: 가슴 | 등 | 하체 | 어깨 | 팔 | 코어
- 단일 선택

**② 세트수 입력**
- 숫자 표시 (기본값 0)

**③ 메모 입력**
- 텍스트 영역
- 플레이스홀더: "운동 메모를 자유롭게 작성하세요..."
- 안내 문구: "운동 전·중·후 언제든 수정 가능"

**④ 하단 버튼**
- "운동 시작" (accent 색상) → 2. 카운트다운 → 운동중 화면

### 상세 내용
- **세트수 입력 방식**: 미정 (추후 확정)
- **운동 시작 최소 조건**: 부위 선택 + 세트수 입력 모두 필수

---

## 2. 카운트다운 → 운동중 화면

| 항목 | 내용 |
|------|------|
| **진입 경로** | 운동 기록 화면에서 "운동 시작" 클릭 |
| **특이사항** | 2단계로 구성 (카운트다운 → 운동중) |

### 2-A. 카운트다운 화면
- 전체화면, 큰 원형 프로그레스
- 중앙에 카운트다운 숫자 (3, 2, 1...)
- 부위 아이콘 + "근력 강화 운동" 텍스트

### 2-B. 운동중 화면
- 실시간 심박수 (큰 숫자 + 하트 아이콘)
- 활동 킬로칼로리 / 총 킬로칼로리 (2열)
- 하단 드로어(Drawer): 위로 올려서 확장 가능
  - 부위 아이콘 + 타이머(MM:SS.ms) + 세트 프로그레스 링
  - 중지 버튼 / 운동 종료 버튼

### 상세 내용
- **카운트다운**: 3, 2, 1 고정 (설정 불가)
- **운동 종료**: 드로어를 올려서 운동 종료 버튼 클릭
- **캘린더 등록**: 운동 종료 시 자동으로 캘린더에 기록 (별도 버튼 없음)
- **드로어 컨트롤**: 중지(일시정지) 버튼 + 운동 종료 버튼 + "저장하지 않고 종료하기" 버튼
- **세션 제한**: 하루 1세션 고정 — 실수 방지를 위해 저장 없이 종료 가능

---

## 3. 운동 결과 화면 (세션 상세)

| 항목 | 내용 |
|------|------|
| **진입 경로** | 운동 종료 후 / 세션 목록에서 세션 카드 클릭 / 캘린더 항목 클릭 |
| **네비게이션** | ← 뒤로가기, 타이틀 날짜 (예: "3월 15일") |

### 구성 요소

**① 세션 헤더**
- 부위 아이콘 + 부위명
- 날짜 + 총 운동 시간

**② 메모**
- 사용자가 작성한 운동 메모 (예: "벤치프레스 무게 올림 80kg 성공")

**③ 운동 세부사항** (2×2 그리드)
- 운동 시간 (H:MM:SS)
- 활동 칼로리 (KCAL)
- 세트수
- 평균 심박수 (BPM)

**④ 심박수 차트**
- 시간 흐름에 따른 심박수 라인 차트
- 시작 ~ 종료 구간
- 평균 심박수 표시

**⑤ 지도**
- 운동 위치 표시 (핀 마커)
- "운동 위치" 라벨

### 상세 내용
- **메모 수정**: 이 화면에서 직접 메모 수정 가능
- **심박수 데이터**: HealthKit 연동으로 가져옴
- **운동 위치**: 운동 시작 시 자동으로 위치 기록

## 4. 메인화면 (요약)

| 항목 | 내용 |
|------|------|
| **탭** | 일지 |
| **타이틀** | "요약" + 오늘 날짜 (예: 3월 15일 일요일) |
| **역할** | 오늘의 운동 현황을 한눈에 보여주는 대시보드 |

### 구성 요소 (위→아래)

**① 오늘의 운동 카드**
- 원형 프로그레스 + 세트 수
- 운동 부위(예: 가슴), 총 세트 수 표시
- 클릭 → 6. 캘린더 페이지

**② 운동시간 / 세트수 카드** (2열)
- 운동시간: 오늘 총 운동 시간 (분)
- 세트수: 오늘 총 세트 수

**③ 세션 카드**
- 가장 최근 세션 요약 (부위, 칼로리, 요일)
- 클릭 → 3. 운동 결과 화면

**④ 운동 카드**
- "근력 강화 운동" 등 운동 타입 표시
- "운동 시작" 버튼
- 클릭 → 1. 운동 기록 화면

**⑤ 이번 주 세션 리스트**
- 이번 주 완료된 세션 목록 (부위, 요일, 시간, 세트수, 칼로리)
- "더보기 →" 클릭 → 5. 세션 목록 화면

**⑥ 하단 버튼**
- "요약 편집" — 메인화면 카드 내 표시 항목 커스터마이징
- "모든 카테고리 보기" — 추가할 수 있는 카드 종류를 보여주고 선택/추가

### 상세 내용
- **빈 상태(운동 안 한 날)**: 카드 레이아웃은 동일하게 유지, 데이터만 0 또는 비어있음으로 표시
- **요약 편집**: 각 카드 내 표시 항목을 커스터마이징하는 기능
- **모든 카테고리 보기**: 메인화면에 추가할 수 있는 카드 종류를 보여주고 선택/추가하는 기능

## 5. 세션 목록 화면 (전체 세션 히스토리)

| 항목 | 내용 |
|------|------|
| **진입 경로** | 메인화면 이번주 세션 "더보기" 클릭 |
| **네비게이션** | ← 뒤로가기, 타이틀 "세션" |

### 구성 요소

**① 필터 탭**
- 모두 | 가슴 | 등 | 하체 | 어깨 (부위별 필터)

**② 월별 그룹 세션 리스트**
- 월 단위로 그룹핑 (예: "2026년 3월")
- 각 세션 카드: 부위 아이콘 + 부위명 + 칼로리(KCAL) + 날짜/요일
- 클릭 → 3. 운동 결과 화면

### 상세 내용
- **데이터 범위**: 전체 세션 히스토리 (과거 포함)
- **스크롤**: 무한 스크롤로 과거 세션 계속 로드
- **필터**: 단일 선택 (한 번에 하나의 부위만 필터링)
- **삭제/편집**: 이 화면에서는 삭제 기능 없음

## 6. 캘린더 페이지

| 항목 | 내용 |
|------|------|
| **진입 경로** | 메인화면 오늘의 운동 카드 클릭 |
| **네비게이션** | ← 뒤로가기, 타이틀 "캘린더" |

### 구성 요소

**① 요일 헤더**
- 월 | 화 | 수 | 목 | 금 | 토 | 일

**② 월별 캘린더 (무한 스크롤)**
- 월 라벨 (예: "3월", "2월", "1월"...)
- 각 날짜마다 원형 프로그레스 링 (운동량 시각화)
- 운동한 날: 링 채워짐 (주황/골드)
- 운동 안 한 날: 빈 링 (어두운 톤)
- 오늘 날짜: 별도 강조
- 운동 세션이 있는 날: 초록 점(dot) 표시
- 클릭 → 3. 운동 결과 화면

### 상세 내용
- **세션 제한**: 하루 1세션 고정 — 날짜 클릭 시 바로 3. 운동 결과 화면으로 이동
- **프로그레스 링 기준**: 운동 여부만 (on/off), 운동량에 따른 차이 없음
- **스크롤**: 현재 월이 상단, 아래로 스크롤하면 과거 월로 이동 (무한 스크롤)

## 7. 운동 템포 페이지

| 항목 | 내용 |
|------|------|
| **탭** | 템포 |
| **타이틀** | "템포" / 부제: "수축·이완 타이밍에 집중하세요" |

### 구성 요소

**① 모드 토글**
- "수축 먼저" | "이완 먼저"

**② 상태 표시**
- 상태 텍스트 (대기 중 / 수축 / 이완 / 휴식)
- 원형 프로그레스 + 현재 세트 숫자
- 진행률: 현재 세트 / 총 세트

**③ 현재 상태 정보** (3열)
- 현재 세트 | 현재 횟수 | 구간

**④ 시간 설정** (2×2 그리드)
- 수축 시간 (초) — +/- 조절
- 이완 시간 (초) — +/- 조절
- 횟수 (회) — +/- 조절
- 세트 (세트) — +/- 조절

**⑤ 휴식 시간 설정**
- 휴식 시간 (초) — +/- 및 ±10 조절

**⑥ 시작 버튼**
- "시작" (accent 색상)

### 상세 내용
- **수축/이완 전환 알림**: 소리(비프음) + 진동(햅틱) 모두 사용
- **휴식 후 세트 전환**: 휴식 시간 끝나면 자동으로 다음 세트 시작
- **운동중 화면 연동**: 없음 — 독립 기능 (운동중 화면과 무관하게 단독 사용)
- **독립 타이머**: 운동중 화면(2-B)의 타이머와 별개로 템포 자체 타이머가 독립적으로 동작 — 운동중에도 템포 탭으로 전환하면 템포 타이머가 따로 흐름

## 8. 프로틴 페이지

| 항목 | 내용 |
|------|------|
| **탭** | 프로틴 |
| **타이틀** | "프로틴" / 부제: "가격 분석 · g당 최저가를 찾아보세요" |

### 구성 요소

**① 카테고리 필터**
- 전체 | WPC | WPI | Blend

**② 프로틴 제품 리스트**
- 각 카드: 제품명 + 가격 등급 뱃지(저점/중간/고점)
- 정보: 카테고리, 용량(g), 가격(원), g당 가격(원)
- 클릭 → 9. 프로틴 상세 페이지

### 상세 내용
- **정렬 기준**: 서버에서 결정 (클라이언트 관여 없음)
- **가격 데이터**: 서버에서 실시간 크롤링
- **가격 등급(저점/중간/고점)**: 서버에서 룰 정의 및 판정

---

## 9. 프로틴 상세 페이지

| 항목 | 내용 |
|------|------|
| **진입 경로** | 프로틴 페이지에서 제품 카드 클릭 |
| **네비게이션** | ← 뒤로가기, 타이틀 "상세 정보" |

### 구성 요소

**① 제품 헤더**
- 제품명 + 가격 등급 뱃지
- 용량 · 카테고리 · 맛

**② 특징 카드**
- 제품 특징 리스트 (예: "인지도가 높음", "맛과 성분은 좋지만 가격이 비싼편")

**③ 현재가 정보**
- 현재 가격 (원) + g당 가격

**④ 가격 추이 차트**
- 시간에 따른 가격 변동 라인 차트
- X축: 날짜 (예: 24-01 ~ 26-0x)
- Y축: 가격 (원)

**⑤ 구매 버튼**
- "구매하러 가기" — 외부 링크

### 상세 내용
- **구매 링크**: 외부 쇼핑몰 링크로 연결 (예: 쿠팡, 아이허브) — 서버에서 URL 제공
- **특징 카드 데이터**: 서버에서 제공 (크롤링 데이터 기반)
- **가격 추이 차트 기간**: 전체 기간 (서버에 데이터가 있는 만큼 모두 표시)

## 10. 잠금화면 위젯

| 항목 | 내용 |
|------|------|
| **타입** | iOS 잠금화면 위젯 |
| **컨셉** | GitHub 잔디(커밋 기록) 스타일 |
| **표시 정보** | 월별 운동 기록을 꽃/도형 아이콘으로 시각화 |
| **부가 정보** | 연간 남은 일수, 달성률(%) 표시 |

### 상세 내용
- **위젯 사이즈**: Medium (가로 직사각형) — 정확한 사이즈는 추후 확정
